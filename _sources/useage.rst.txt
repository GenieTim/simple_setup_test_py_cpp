Useage
======

After :doc:`installing <installation>` this package, you can import it in Python like so:

.. code:: python

  import simple_setup_test_py
  import simple_setup_test_py_cpp

Then, follow the help (:code:`help(simple_setup_test_py); help(simple_setup_test_py_cpp);`), 
take a look at the examples used for `tests <https://github.com/GenieTim/simple_setup_test_py/tree/main/tests>`_,  
or the documentation of the module as well as the tests to see what you can do with the library.
